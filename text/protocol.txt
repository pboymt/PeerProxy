Message Structure:

Message Header:
+--------+-------------------------------+-------------------
| Scope  |        Message Header         | Typed Message
+--------+--------------+----------------+-------------------
| Name   | Message Type | Message Length |
+---+----+--------------+----------------+-------------------
| Length |      1       |        4       |
+---+----+--------------+----------------+-------------------

Message Type
0x00 Raw Message
0x01 Text Message
0x02 JSON Message
0x03 Simple Buffer
0x10 Proxy Message
